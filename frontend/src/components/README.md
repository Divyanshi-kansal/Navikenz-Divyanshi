# Components folder

The `components` folder contains reusable React components used throughout the frontend application.



## Responsibilities

- Build reusable UI elements
- Display dynamic data received from Backend.

---

### Files

`ChatForm.jsx/`
Provides the input area where users can type and send messages to the chatbot.

`ChatSidebar.jsx/`
Displays all chat sessions and allows users to switch between conversations, delete chats, and manage sessions.

`MessageFeed.jsx/`
Displays the complete conversation between the user and the AI assistant.
Also Renders Markdown Responses returned by the backend.

`LoginForm.jsx/`
Provides the user login interface and submits authentication requests to the backend.

`SignupForm.jsx/`
Provides the user registration interface and validates user details before sending them to the backend.
