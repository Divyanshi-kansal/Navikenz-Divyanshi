# src folder

Contains main source code of the application.

---

## Folder Structure

`components/`
Contains reusable UI components.

Example:
- ChatForm
- ChatSidebar
- LoginForm
- MessageFeed
- SignupForm

`pages/`
Contains complete application pages.

Example:
- ChatPage
- LoginPage
- SignupPage

`services/`
Contains functions responsible for communicating with the backend APIs.

Examples:
- User Login
- User Registration
- Send Chat Messages
- Fetch Chat History

`App.jsx/`
Root component of the application.

`App.css/`
Styling of main application.

`main.jsx/`
Entry point of the React application.

`index.css/`
Global styles applied throughout the application.
