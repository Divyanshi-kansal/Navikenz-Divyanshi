# Pages

The pages folder contains the main page of the application.

## Responsibilities

- Display complete application views

---

### Files

`ChatPage.jsx/`
Serves as the main chatbot interface. It manages chat sessions, displays conversations, communicates with the backend APIs, and coordinates all chat-related components.

`LoginPAge.jsx/`
Displays the login page and authenticates existing users.

`SignupPage.jsx/`
Displays the user registration page and allows new users to create an account.
